# LLM 기반 AI 학습 튜터 서비스

LLM(Google Gemini)을 활용해 사용자의 질문에 실시간으로 답변하고 학습을 돕는 대화형 튜터 웹 서비스입니다. Ruby on Rails 기반 REST API 서버와 React 프론트엔드, JWT 인증, 멤버십/결제, 채팅 스트리밍 등 실제 서비스에 가까운 풀스택 아키텍처를 직접 설계 및 구현했습니다.
