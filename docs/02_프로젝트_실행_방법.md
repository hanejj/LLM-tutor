## 프로젝트 실행 방법

### 0. Docker Compose로 통합 실행

루트 디렉토리에서 백엔드, 프론트엔드, MySQL을 한 번에 실행할 수 있습니다.

```bash
cd ai-tutor

# 환경 변수(.env) 파일 설정
# 템플릿 파일을 .env로 복사한 후, 값을 설정하세요.

# Windows CMD
copy config\env.example .env
# Mac/Linux
cp config/env.example .env

# Gemini API Key 발급 방법 안내
1. [Google AI Studio](https://makersuite.google.com/app/apikey)에 접속하세요.
2. "Create API Key" 혹은 "Get API Key" 버튼을 클릭하세요.
3. 새로운 프로젝트를 생성하거나, 기존 프로젝트를 선택합니다.
4. 발급된 API 키를 복사하세요.

# .env 파일에 키 등록
.env 파일을 열어서 아래와 같이 복사한 키 값을 입력하세요:
GEMINI_API_KEY=여기에_복사한_API_키_붙여넣기

# 프로젝트 터미널에서
docker-compose up --build
```

- 프론트엔드: `http://localhost:3001`
- 백엔드 API: `http://localhost:3000/api/v1`

---

### 1. 실행 확인

1. **Backend**: http://localhost:3000 → rails 페이지 표시
2. **Frontend**: http://localhost:3001 → 로그인 페이지 표시
3. **테스트 계정**:
  - 이메일: `user1@example.com` / 비밀번호: `password123` (베이직)
  - 이메일: `user2@example.com` / 비밀번호: `password123` (프리미엄)
  - 이메일: `user3@example.com` / 비밀번호: `password123` (체험)
  - 이메일: `user4@example.com` / 비밀번호: `password123` (멤버십 없음)
