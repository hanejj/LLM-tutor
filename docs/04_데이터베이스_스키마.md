## 데이터베이스 스키마

### Memberships (멤버십)
| 컬럼 | 타입 | NULL | 기본값 | 설명 |
|------|------|------|--------|------|
| id | bigint | NOT NULL |  | PK |
| name | string |  |  | 멤버십 이름 (베이직, 프리미엄 등) |
| features | string |  |  | 포함 기능 (쉼표 구분) |
| expires_at | datetime |  |  | 만료일 |
| coupon_count | integer | NOT NULL | 0 | 멤버십 구매 시 지급되는 기본 채팅 쿠폰 수 |
| created_at | datetime | NOT NULL |  |  |
| updated_at | datetime | NOT NULL |  |  |

- 인덱스: 없음

### Users (사용자)
| 컬럼 | 타입 | NULL | 기본값 | 설명 |
|------|------|------|--------|------|
| id | bigint | NOT NULL |  | PK |
| email | string |  |  | 이메일 (DB 레벨 유니크 인덱스 없음) |
| password_digest | string |  |  | 암호화된 비밀번호 |
| membership_id | bigint | NULL 허용 |  | FK → memberships.id |
| chat_coupons | integer | NULL 허용 |  | 채팅 쿠폰 개수 |
| created_at | datetime | NOT NULL |  |  |
| updated_at | datetime | NOT NULL |  |  |

- 인덱스: `index_users_on_membership_id`
- 외래 키: `users.membership_id` → `memberships.id`
